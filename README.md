# APX Labs - Modern Web Development Agency

A professional agency landing page built with HTML, CSS, and vanilla JavaScript. Showcasing services, technology stack, process, portfolio, and contact information.

## 🚀 Features

- **Modern UI/UX**: Clean, modern design with smooth animations and micro-interactions
- **Responsive Design**: Fully responsive layout for all devices (desktop, tablet, mobile)
- **Dark Mode**: Built-in theme toggle with localStorage support
- **Smooth Animations**: Scroll animations using Intersection Observer
- **Interactive Elements**: Parallax scrolling, hover effects, dynamic backgrounds
- **Navigation**: Sticky navbar with scrollspy and active states
- **Progress Tracking**: Scroll progress bar and reading progress indicator
- **Portfolio Grid**: Filterable portfolio section with modal previews
- **Tech Stack Showcase**: Interactive display of technologies with icons
- **Contact Form**: Working contact form with client-side validation
- **SEO Optimized**: Proper meta tags, semantic HTML, and fast loading
- **Accessibility**: ARIA labels, keyboard navigation, focus management
- **Loader**: Custom loading animation on page load

## 🚀 Getting Started

### Prerequisites

- Modern web browser (Chrome, Firefox, Safari, Edge)
- Text editor or IDE (VS Code recommended)

### Installation

1. **Clone the repository** (if applicable)
2. **Download the files**
3. **Open `index.html`** in your web browser

### Usage

- Navigate through the sections using the navbar
- Click "Get Free Consultation" or "Start Your Project" to open the contact form
- Toggle dark/light mode using the sun/moon icon in the navbar
- Hover over portfolio items to see details
- Click on portfolio items to view larger previews
- Use the theme toggle to switch between dark and light mode
- Scroll through the page to see animations and progress tracking
